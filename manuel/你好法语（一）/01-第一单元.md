## 动词的直陈式

直陈式现在时是动词变位的基础。直陈式，即对实际存在的动作或事实进行陈述或描述；现在时，即说话者陈述的事实或动作发生在当下或者最近。

# 第2课

## L'article défini 定冠词

un/une 是不定冠词。

* le/la(类似于英语中的 the，注意元音或哑音h):
  - Le Français, La Française;
  - le étudiant(两个元音相邻，省音) -> l'étudiant; l'étudiante
  - le homme -> l'homme

### 用法

* 表示前面已经提及的人、物：C'est une belle voiture. La voiture est à M. Dupont.
* 谈话双方皆知的人、物：Nous n'avons pas de cours ce matin. Le professeur est malade.
* 用在受其它成分限定的名词前：Anne est la petite sœur de Paul.
* 人、物的总体概念：J'aime beaucoup le café.
* 独一无二的事物：La Terre est ronde.

## La forme féminine des noms 名词的阴阳性

基本规则之一：+ e：étudiant -> étudiante
基本规则之二：以e结尾的不变：le secrétaire -> la secrétaire

特殊规则：
一：-teur -> -trice：directeur -> directrice
二：-eur -> -euse：danseur -> danseuse
三：-er -> -ère: boulanger -> boulangère
四：-f -> -ve: juif -> juive
五：-ien -> -ienne: informaticien -> informaticienne

历史原因：
某些职业只有阳性形式，为了指称女性，可以使用 femme 辅助：la femme professeur

chemise 是男式衬衫，阴性名词；chemisier 是女式衬衫，阳性名词。

## Préposition + nom de pays/ville 介词+国名/城市名

国家具有阴阳性，但城市没有。国家阴阳性与该国家国民的气质无关，常见国家里可通过词形判断：
* 以e结尾的词：除了 Cambodge、Mexique 之外都是阴性的，如 France、Chine、Espange。
* 其它为阳性：Japon、Portugal

### 介词+国名/城市名

表示“在”、“去”的意思时：

A. 阴性国名，用 en：en Chine，en Allemagne
B. 阳性国家，用 au：au Japon, au Canada
C. 阳性国家以元音开头，用 en：en Iran；
D. 城市：用 à

au 是 à + le，是一种缩合冠词。

* ton/ta/tes
* son/sa/ses

### 连读

在以元音或哑音 h 开头的单数阴性名词或形容词前，不用 ma/ta/sa，依然使用 mon/ton/son，以方便发音：

* mon école, ton amie, son histoire.
